# Introduction

This documentation will guide you on how to use the Contest Hive API.

## What is Contest Hive?

A simple web app that keeps track of all the contests happening on various competitive programming platforms.

## Platforms

Contest Hive supports 7 online judge platforms. See the list below.

- [Atcoder](https://atcoder.jp/)
- [Codechef](https://www.codechef.com/)
- [Codeforces](https://codeforces.com/) ( CF GYM is also available )
- [Hackerearth](https://www.hackerearth.com/)
- [Hackerrank](https://www.hackerrank.com/)
- [Leetcode](https://leetcode.com/)
- [Toph](https://toph.co/)
